> 17track golang版本SDK
> 官方文档请参考[api文档](https://console.17track.net/zh-cn/doc)

### Install

```bash
go get github.com/wksw/17track-golang-sdk
```

### Install 17track command

```bash
go get github.com/wksw/17track-golang-sdk/cmd/17track
```

### Get Latest carriers and countries

```bash
go generate
```
